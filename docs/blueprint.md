# **App Name**: PisoMate

## Core Features:

- Philippine Loan Dashboard: View total business liquidity, total number of active clients, and a searchable scrollable list of client loan cards in PHP currency.
- Smart 10-Step Scheduler: Automatically generates an editable 10-installment payment checklist for every new client loan entry.
- One-Tap Payment Logging: Modal-driven logging to mark installments as paid, record the specific payment date, and automatically subtract from the outstanding balance.
- Zero-Loss Protection Logic: Calculates remaining balances in real-time and prevents the account balance from falling below zero to ensure accounting accuracy.
- Collection Advisor Tool: A generative AI tool that analyzes a client's specific payment schedule behavior to suggest personalized collection approaches.
- Local JSON Export: Generate and download a backup of all local lending data into a JSON file format for offline data protection.
- Offline Local Storage: Uses persistent browser-based database technology to ensure all lending records remain on the device without a login or cloud connection.

## Style Guidelines:

- Primary color: Professional Cobalt Blue (#3B5998), evoking trust and financial stability.
- Background color: Cool Gray-White (#F2F4F7) for a clean, minimalist high-contrast readability against dark text.
- Accent color: Sky Cerulean (#5DADE2) used to distinguish checkboxes and active 'Add Client' actions.
- The font 'Inter' is used throughout for its geometric precision and superior legibility at varying scales, perfect for numeric balance sheets.
- Bold, stroke-based Material Design icons to indicate adding, deleting, and checking items to ensure a fast, functional UI.
- A focused, single-column mobile-first layout with high-padding cards to provide plenty of touch surface for small business owners.
- Quick, responsive slide transitions between the client list and the individual record screens for an app-like experience.
